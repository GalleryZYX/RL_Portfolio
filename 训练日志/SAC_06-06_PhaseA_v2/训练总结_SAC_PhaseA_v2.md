# SAC Phase A v2 训练总结：折中参数调优

**日期：** 2026-06-06  
**编写人：** 胡锦宏、刘骏哲元  
**关联训练：** `python scripts/train_sac_portfolio.py`（130K 步早停）

---

## 一、背景

Phase A 成功消除过拟合，但 [64,32] 网络容量不足导致欠拟合（训练 -2.37%）。Phase A v2 目标：在网络容量和泛化能力之间找到平衡。

---

## 二、参数变更

| 参数 | Phase A | Phase A v2 | 理由 |
|------|:------:|:------:|------|
| `--net-arch` | `64,32` (~4万参数) | **`128,64`** (~10万参数) | 恢复适度容量 |
| `--ent-coef` | 0.05 | **0.01** | 降低探索，允许收敛 |
| `--learning-rate` | 1e-3 | **3e-4** | 更稳定的梯度更新 |
| 其他 | 不变 | 不变 | — |

**额外修复：** 早停 callback 从 `model.stop_training=True` 改为 `return False`，训练在 130K 步成功截断。

## 六、结论

1. **[128,64] + ent_coef 0.01 是三版中最佳平衡点**：训练收益首次转正（+2.95%）、过拟合不存在、测试收益为正（+0.46%）
2. **Q1/Q2 分化是三版共有模式**：Q1 ~-8%、Q2 ~+7.5% 夏普 >3，说明模型在特定市场环境下能找到有效信号
3. **量价特征可能已到信息上限**：三版测试收益都在 -0.3%~+0.7% 窄幅波动，需要 Phase B 的新闻/舆情特征来突破
4. **早停修复验证成功**：130K 正常截断，节省了 57% 的训练时间
