# SAC Phase B 训练总结：大盘+截面舆情特征

**日期：** 2026-06-07  
**编写人：** 胡锦宏、刘骏哲元  
**关联训练：** `python scripts/train_sac_portfolio.py`（270K 步早停）

---

## 一、背景

Phase A v2 实现最佳参数平衡后，量价特征可能已达信息上限。Phase B 目标：新增大盘和截面排名特征，尝试突破性能瓶颈。教师建议的新闻/舆情方向中，AKShare 热门排名仅支持实时数据（无历史），Tushare 新闻情感因 API 限速暂未采集；本次先接入从现有数据衍生的大盘+截面特征。

---

## 二、新增特征

### 2.1 特征管线

新建 `data/sentiment_pipeline.py`，提供三类特征生成能力：

| 类别 | 特征 | 来源 | 说明 |
|------|------|------|------|
| 大盘 | `market_breadth` | 现有 CSV 计算 | 当日上涨股票占比 (0~1) |
| 大盘 | `market_return` | 现有 CSV 计算 | 等权平均涨跌幅 |
| 大盘 | `market_vol` | 现有 CSV 计算 | 截面涨跌幅标准差 |
| 截面 | `pct_chg_rank` | 现有 CSV 计算 | 涨跌幅在股票池内百分位 |
| 截面 | `turnover_rate_rank` | 现有 CSV 计算 | 换手率百分位 |
| 截面 | `pe_ttm_rank` | 现有 CSV 计算 | 市盈率百分位（越低越高） |
| 情感 | `news_count` | Tushare major_news | 当日新闻数量（暂未采集，填 0） |
| 情感 | `sentiment_mean` | Tushare + SnowNLP | 当日情感均值（暂未采集，填 0.5） |

## 六、结论

1. **新增 6 个特征未带来显著提升**：测试收益 +0.45% vs v2 的 +0.46% 几乎持平，验证奖励 0.246 vs 0.249 略低
2. **量价特征已达信息上限**：四版测试收益始终在 -0.3%~+0.7% 窄幅波动，无论特征多少、网络大小，OOS 表现无本质突破
3. **Q1/Q2 分化模式极其稳健**：四版 Q1 均 ~-8%、Q2 均 ~+7.5% 夏普 >3，说明模型确实学到了可泛化的市场环境识别能力
4. **需要外部数据突破**：大盘+截面特征只是量价信息的重组，真正的增量信息需来自新闻情感、热门排名等外部数据源
5. **`sentiment_pipeline.py` 已预留 Tushare 新闻接口**：`fetch_sentiment=True` 即可采集，但受 API 限速需约 1 小时
