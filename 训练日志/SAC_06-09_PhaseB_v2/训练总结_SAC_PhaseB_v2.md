# SAC Phase B v2 训练总结：价格隐含情感代理特征

**日期：** 2026-06-09  
**编写人：** 胡锦宏、刘骏哲元  
**关联训练：** `python scripts/train_sac_portfolio.py`（130K 步手动终止）

---

## 一、背景

Phase B 尝试的 Tushare 新闻情感采集因 API 限速（1次/小时）不可行。Phase B v2 换思路：隔夜跳空本身就是市场对盘后信息（含新闻）的消化结果——这是嵌入在价格中的情感代理变量。无需外部 API。

---

## 二、新增特征

### 2.1 价格隐含情感代理

| 特征 | 公式 | 含义 |
|------|------|------|
| `overnight_gap_abs` | abs(open - pre_close) / pre_close | 隔夜信息冲击的幅度（情感强度） |
| `intraday_momentum` | (close - open) / open | 日内动量方向（市场对盘前信息的消化方向） |

### 2.2 特征集变化

```
Phase B (18):   Phase A v2(12) + market_breadth, market_return, market_vol,
                pct_chg_rank, turnover_rate_rank, pe_ttm_rank

Phase B v2(14): Phase A v2(12) + overnight_gap_abs, intraday_momentum
                （去掉无效的 6 个大盘/截面特征）
```

观测维度：663 → 765。特征由环境 `_prepare_data` 自动计算，无需生成增强 CSV。

## 五、结论

1. **价格隐含情感代理特征无效**：overnight_gap_abs 和 intraday_momentum 未能提升验证奖励（0.243 vs v2 的 0.249）
2. **Phase A v2 保持最优**：12 个核心量价特征 + [128,64] + ent_coef 0.01，测试 +0.46% 夏普 0.16
3. **从价格衍生更多特征已到尽头**：五轮训练、11~18 特征、4 万~70 万参数，测试收益始终在 ~0% 附近
4. **仅剩两条突破路径**：
   - **外部数据**（新闻/舆情/宏观）：需付费 API 或更高频 Tushare 积分
   - **方法论升级**（改造奖励函数、Walk-Forward）：Phase C
