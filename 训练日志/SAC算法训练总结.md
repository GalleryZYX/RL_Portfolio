# SAC 算法训练总结

**日期：** 2026-06-10  
**编写人：** 胡锦宏、刘骏哲元  
**用途：** 团队汇报

---

## 一、项目概述与目标

使用 **SAC（Soft Actor-Critic）强化学习算法**，训练智能体管理 51 只中国 A 股的投资组合。智能体每天观察市场数据，决定资金在股票间的分配比例，目标是在扣除交易成本后实现最大化的风险调整收益。

**数据规模：** 51 只股票，覆盖 2020-2026 年，共 1529 个交易日  
**环境：** 基于 Gymnasium 的 `PortfolioEnv`，612~936 维观测空间，51~52 维连续动作空间  
**算法：** Stable Baselines3 SAC，MLP 策略网络

## 二、实验全景：十轮训练，四次 Walk-Forward

### 2.1 实验演进树

```
Phase 0: [256,256] + auto熵系数 → 严重过拟合（训练 178x vs 测试 -0.29%）
  │
Phase A: [64,32] + 固定熵 0.05 + L2 + 验证早停 + 12特征 → 欠拟合
  │
Phase A v2: [128,64] + 熵 0.01 + lr 3e-4 → ★ 单窗口最佳
  │
  ├─ Phase B: +6 大盘截面特征 → 无效
  │   └─ Phase B v2: 情感代理 + Tushare 新闻尝试 → 终止
  │
  ├─ Phase C3: 风险惩罚奖励函数 → 无效
  ├─ Phase C4: 市场状态特征（均线偏离 + 波动率区间）→ 边际改善
  │
  ├─ Walk-Forward: 5 窗口滚动回测 → ★ 揭示真实水平
  │
  └─ Phase D: 现金资产 + 大盘趋势信号 → 无帮助
  │
  └─ Phase E: 移植 PPO 奖励分量（波动率/夏普/换手率）→ 边际改善 +0.2%
```

|------|:--:|------|------|

| Phase A | 12 | gap, intraday_amp, close_loc（替换 vol, amount） | 全部尺度不变 |

## 三、核心发现

### 3.1 过拟合 → 欠拟合 → 平衡：Phase 0 ~ A v2

| 指标 | Phase 0 | Phase A | **Phase A v2** |
|------|:------:|:------:|:------:|
| 网络 | [256,256] | [64,32] | **[128,64]** |
| 熵系数 | auto→0.00008 | 0.05 | **0.01** |
| L2 正则 | 无 | 有 | 有 |
| 训练收益 | +17,700% | -2.37% | **+2.95%** |
| 测试收益 | -0.29% | +0.66% | **+0.46%** |
| 过拟合程度 | 严重 | 欠拟合 | **平衡** |

**关键教训：**
- `ent_coef=auto` 会降至近乎零，策略坍缩为确定性（过拟合根因）
- `[256,256]` 约 70 万参数（vs 714 天数据）严重过参数化
- 时序验证集 + 早停是防止过拟合的最有效手段

### 3.2 特征工程的边界：Phase B ~ D

| 改进方向 | 代表实验 | 测试收益 | 是否有效 |
|------|------|:------:|:--:|
| 从价格衍生 | Phase B v2 （隔夜跳空） | 未完成 | ✗ 终止 |
| 大盘+截面 | Phase B （6 特征） | +0.45% | ✗ 无效 |
| 风险惩罚 | Phase C3 （回撤+集中度） | +0.43% | ✗ 无效 |
| per-stock 趋势 | Phase C4 （均线偏离） | +0.51% | △ 边际 |
| 现金资产 | Phase D | — | ✗ 无效 |
| 大盘趋势 | Phase D | — | ✗ 无效 |
| **波动率惩罚** | **Phase E** | **+0.67%** | **△ 边际** |
| **夏普奖励** | **Phase E** | — | **△ 边际** |
| **换手率惩罚** | **Phase E** | — | **△ 边际** |

**结论：在现有数据框架内，从量价数据衍生特征的改进空间已耗尽。** 十一到十七个特征，单窗口 OOS 始终在 0.4%~0.7%。奖励函数改造带来 +0.2% 边际改善，不改变基本结论。

### 3.3 Walk-Forward 揭示的真相

这是项目最重要的发现。单次测试（只看 2026 年）给出的结论是"策略几乎没用"（+0.5%），但 Walk-Forward 五窗口回测揭示了完全不同的故事：

#### 四次 Walk-Forward 方案终局对比

| 测试年 | 12特征（原始） | 15特征（C4） | +现金 | +大盘趋势 |
|:--:|:------:|:------:|:------:|:------:|
| **2022** | -18.81% | -18.88% | -18.54% | -18.51% |
| 2023 | -6.01% | **-5.44%** | -5.89% | -5.55% |
| 2024 | +14.00% | +13.54% | +13.28% | +13.23% |
| **2025** | +25.96% | **+27.94%** | +26.16% | +25.70% |
| 2026 | +0.79% | +0.56% | +0.23% | +0.16% |
| **均值** | +3.18% | **+3.54%** | +3.05% | +3.01% |

#### 汇总统计

| 指标 | 最优方案（C4） |
|------|:------:|
| 年均收益率 | **+3.54%** |
| 收益标准差 | ±17.94% |
| 年均夏普比率 | 0.22 |
| 正收益年份占比 | **60%（3/5 年）** |
| 最佳单年 | 2025: +27.94%，Sharpe 1.49 |
| 最差单年 | 2022: -18.88%，Sharpe -0.89 |

## 五、技术贡献

### 5.1 训练框架

- **时序验证早停**：`validation_callback.py`，按时间切分训练/验证集，验证不提升自动终止
- **VecNormalize 训练/评估隔离**：独立实例防止归一化统计量污染
- **模块化特征管线**：`sentiment_pipeline.py` 支持大盘、截面、情感特征生成
- **Walk-Forward 回测框架**：`walk_forward.py`，多窗口自动化训练+评估

### 5.2 经验教训

| 教训 | 详情 |
|------|------|
| `ent_coef=auto` 的陷阱 | 在有限数据上 α → 0，策略坍缩为确定性过拟合 |
| 时序切分不可随机 | 金融数据的未来信息泄露会致命歪曲验证结果 |
| 单点测试不可靠 | 2026 年 +0.5% 的结论被 Walk-Forward 彻底推翻 |
| 特征工程有上限 | 从 11 到 17 特征，均值始终 ~3.5% |
| 两年训练不够学择时 | 现金资产在 380 天数据上无法学会牛熊判断 |

## 六、限制与展望

### 6.1 当前限制

1. **训练数据量有限**：每个 WF 窗口仅 ~380 个交易日（约 1.5 年），不足以覆盖完整牛熊周期
2. **特征均基于量价**：缺乏新闻情感、宏观经济、资金流向等外部数据
3. **模型架构简单**：MLP 无法捕获时序依赖，也不支持多时间尺度特征
4. **交易约束简化**：不包含涨跌停、T+1、流动性限制等 A 股实际约束

### 6.2 突破路径

| 方向 | 具体方案 | 优先级 |
|------|------|:--:|
| 更长的数据 | 扩展至 2015 年，覆盖 2 个完整牛熊周期 | ★★★ |
| 外部数据 | 付费 Tushare 积分获取新闻/宏观数据 | ★★★ |
| LSTM 策略网络 | 用循环网络替代 MLP，捕获长期时序依赖 | ★★☆ |
| 分层强化学习 | 上层选市场状态 → 下层选具体股票 | ★★☆ |
| Transformer | 学习跨股票的关系和注意力模式 | ★☆☆ |

## 七、汇报要点（一分钟版）

> 我们用强化学习训练了一个 A 股投资组合管理模型。核心发现是：**策略在牛市中表现优秀（2025 年 +28%，夏普 1.49），但熊市中无法自保（2022 年 -19%）**。五年 Walk-Forward 回测显示年均收益 +3.5%，60% 的年份正收益。这是基于纯量价数据能提取的几乎全部信息。下一步需要更长的历史数据或接入外部信息源（新闻、宏观）来突破熊市防御这个瓶颈。
